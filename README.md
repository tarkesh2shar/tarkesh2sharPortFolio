# Tushar Pant Portfolio

A modern portfolio website for Tushar Pant (`tarkesh2shar`), built with Vite, React, TypeScript, and Three.js.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
